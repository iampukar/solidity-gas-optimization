## Multiple Require

Using multiple require statements is cheaper than using a require statement with multiple check combinations.

There are more advantages, like easier code readability and better coverage reports.

### Preview

MultipleRequire.badRequire()
execution cost 2953 gas (Cost only applies when called by a contract)

MultipleRequire.goodRequire()
execution cost 2951 gas (Cost only applies when called by a contract)

### Guidelines

Use multiple require(s)
